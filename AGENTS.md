# AGENTS.md

This repo ships customer-facing UI, so frontend polish matters as much as correctness.

## UI Skill

Use `$userinterface-wiki` for any task that touches:

- visual design or layout
- motion, transitions, or animation timing
- typography, spacing, or iconography
- interaction design, perceived performance, or UX flow
- component styling, hover/press states, or responsive behavior

For non-trivial UI work, consult the skill twice:

1. before implementation, to shape the plan
2. before handoff, to run a focused UI self-review

When the skill materially affects the solution, mention the relevant categories in your summary or workpad notes, for example `animation`, `typography`, `laws of UX`, or `visual design`.
