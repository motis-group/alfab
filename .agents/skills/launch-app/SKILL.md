# Launch App

1. Install dependencies if they are not already present:
   `npm install`
2. Start the app from the repo root:
   `npm run dev`
3. Wait for Next.js to report the listening URL. Prefer `http://127.0.0.1:3000`, and if port `3000` is occupied, use the fallback port shown in the command output.
4. Use the running dev server for browser-based validation, then stop it when validation is complete.
